# Object-Oriented Programming (OOP) in C++

This repository contains my C++ implementations of Object-Oriented Programming concepts, demonstrating how classes and objects are used to build modular, reusable, and maintainable programs. It also includes beginner-friendly practice questions based on real-world scenarios.

---

### Topics Covered

- Classes & Objects
- Properties & Methods
- Access Modifiers
- Encapsulation
- Constructors
- Parameterized Constructors
- Constructor Overloading
- Copy Constructors
- `this` Pointer
- Setter & Getter Functions
- Deep Copy vs Shallow Copy
- Dynamic Memory Allocation using Pointers

---

### Concepts Implemented

| Concept | Description |
|---------|-------------|
| Classes & Objects | Creating custom classes and objects |
| Public & Private Members | Understanding data hiding |
| Setter & Getter | Controlled access to private data |
| Constructors | Object initialization |
| Non-Parameterized Constructor | Default object creation |
| Parameterized Constructor | Initializing objects with values |
| Constructor Overloading | Multiple constructors in one class |
| `this` Pointer | Referring to the current object |
| Copy Constructor | Creating copies of objects |
| Deep Copy | Independent copy of dynamically allocated memory |
| Shallow Copy | Reference-based object copying |
| Dynamic Memory | Using pointers inside classes |

### Important Concepts

#### Access Modifiers

| Modifier | Accessibility |
|----------|---------------|
| `public` | Accessible from anywhere |
| `private` | Accessible only inside the class |
| `protected` | Accessible inside the class and derived classes |
