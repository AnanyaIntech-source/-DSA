# Strings in C++

This folder contains beginner-friendly C++ programs demonstrating string input, output, manipulation, and commonly used string functions. These programs build a strong foundation for solving string-based Data Structures & Algorithms problems.

---

### Topics Covered

- String declaration
- Reading single-word strings
- Reading multiple strings
- Reading multiple words using loops
- `getline()` function
- Checking for empty strings
- String length using `size()`
- Comparing strings
- Copying strings
- Creating strings using constructors

---

### Programs Included

| Program | Concept |
|---------|---------|
| Read First Word | Input a single word using `cin` |
| Read Two Words | Input and print two strings |
| Read Multiple Words | Read strings continuously until EOF |
| Read Complete Line | Input full sentences using `getline()` |
| Empty String Check | Check whether a string is empty |
| String Length | Find the length of a string using `size()` |
| Compare Strings | Compare multiple strings using relational operators |
| Copy String | Copy one string into another |
| Create New String | Initialize a string with repeated characters |

---

### String Functions Used

| Function | Purpose |
|----------|---------|
| `getline()` | Reads an entire line including spaces |
| `empty()` | Checks whether the string is empty |
| `size()` | Returns the number of characters in the string |

---

### Key Learning

Through these programs, I learned:

- Different ways to take string input
- The difference between `cin` and `getline()`
- Basic string manipulation
- Frequently used string member functions
- String comparison and initialization techniques

These concepts form the foundation for advanced string algorithms such as palindrome checking, anagrams, substring searching, and pattern matching.
