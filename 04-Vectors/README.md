This folder contains beginner to intermediate C++ programs demonstrating the use of vectors (dynamic arrays), STL vector functions, common vector operations, and practice problems frequently encountered in Data Structures & Algorithms.

- Searching
- Reversing Vectors
- Copying Vectors
- Merging Vectors
- Problem Solving using Vectors

---

### STL Functions Used

| Function | Purpose |
|----------|---------|
| `push_back()` | Insert an element at the end |
| `pop_back()` | Remove the last element |
| `insert()` | Insert one or more elements |
| `size()` | Returns the number of elements |
| `capacity()` | Returns the allocated storage capacity |
| `front()` | Returns the first element |
| `back()` | Returns the last element |
| `begin()` | Iterator pointing to the first element |
| `end()` | Iterator pointing after the last element |
| `sort()` | Sort elements in ascending order |
| `reverse()` | Reverse the order of elements |
| `swap()` | Exchange two elements |
